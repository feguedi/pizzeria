# Pizzería

Proyecto de pizzería para personalizar las órdenes y lo demás...

## Objetivo

Ninguno. Solo quiero armar una interfaz para hacer pizzas.

## Instalar

`$ npm install`

## Construir para producción

`$ npm run build`

## Iniciar

`$ npm start`

## Dependencias

### Cliente

- React
- D3.js
- Chakra-ui

### Servidor

- Hapi.js
- MongoDB
- JSON Web Tokens
